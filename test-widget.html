ID Fine Widget Test Sayfası

Bu sayfa, ID Fine chat widget'ının test edilmesi için oluşturulmuştur.

Sağ alt köşede ID Fine logosunu görmelisiniz. Tıklayarak sohbet penceresini açabilirsiniz.

Widget özellikleri:

  • Shadow DOM ile stil izolasyonu
  • WebSocket ile gerçek zamanlı streaming
  • REST API fallback
  • Markdown rendering
  • Mobil uyumlu tasarım
